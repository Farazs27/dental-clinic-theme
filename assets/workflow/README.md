# Smile Makeover Workflow Images

This folder contains images for the smile makeover/aesthetic analysis workflow page.

## Folder Structure

The workflow page (`esthetische-analyse.html`) uses images from these numbered folders:

```
1. Eerste consult & documentatie/   -> First consultation images
2. Studie & Ontwerp/                -> Study & design images
3. Preparatie.../                   -> Preparation images
4. Tijdelijke restauraties/         -> Temporary restorations
5. Definitieve plaatsing.../        -> Final placement
6-eindresultaat/                    -> Final result
```

## What Images to Add

For each step of your smile makeover workflow, add relevant images:

### Step 1 - First Consultation
- Patient documentation photos
- Initial smile photos
- Before photos

### Step 2 - Study & Design  
- Digital smile design mockups
- Wax-up photos
- Treatment planning images

### Step 3 - Preparation
- Tooth preparation images
- Clinical photos during treatment

### Step 4 - Temporary Restorations
- Temporary veneers/crowns photos
- Trial smile photos

### Step 5 - Final Placement
- Bonding process images
- Final restoration placement

### Step 6 - Final Result
- After photos
- Final smile photos
- Before/after comparisons

## Image Guidelines

1. **Patient Consent**: Ensure you have written consent from patients for any photos used
2. **Quality**: Use high-resolution clinical photos (minimum 1200px width)
3. **Format**: PNG or JPG format
4. **Naming**: Keep file names simple (1.png, 2.png, etc.)

## Alternative: Stock Images

If you prefer not to use real patient photos, you can use dental stock images from:
- [Unsplash](https://unsplash.com/s/photos/dental-treatment)
- [Pexels](https://pexels.com/search/dental/)
- [Shutterstock](https://shutterstock.com) (paid)

## Removing This Section

If you don't want to show a workflow on your site, you can:
1. Remove the `esthetische-analyse.html` page
2. Remove the link from the navigation in all HTML files
